# Customer Purchase Analysis

## 📊 Project Overview
This project analyzes customer purchasing behavior using SQL, Python, and Power BI.

## 🛠 Tools Used
- SQL (MySQL)
- Python (Pandas, NumPy)
- Power BI

## 📌 Objectives
- Analyze customer spending patterns
- Perform customer segmentation
- Identify top-performing products
- Study impact of discounts and subscriptions

## 📊 Dashboard Insights
- Revenue by category
- Customer segmentation
- Subscription analysis
- Age group contribution

## 🚀 Key Features
- Used SQL aggregations, subqueries, CTEs, and window functions
- Cleaned and processed data using Python
- Built interactive dashboard in Power BI
